## 🚀 使用方式

### 超簡單!直接打開
1. 解壓縮 `yiru-website-html.zip`
2. 雙擊 `index.html` 即可在瀏覽器中查看
3. 點擊導航連結可以切換頁面

### 使用本地服務器(推薦)
如果您有Python:
```bash
python -m http.server 8000
```

如果您有Node.js:
```bash
npx http-server -p 8000
```

然後打開瀏覽器訪問: http://localhost:8000

## 🔧 客製化修改

### 修改顏色
編輯 `css/style.css` 檔案中的 `:root` 變數:
```css
:root {
  --primary-color: #B83A3C;  /* 主色調 */
  --primary-dark: #8B2729;   /* 深色調 */
  --primary-light: #D44547;  /* 淺色調 */
}
```

### 修改Logo
替換 `images/logo.png` 即可

### 修改內容
直接編輯對應的HTML檔案即可
